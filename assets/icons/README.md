# Icons

This folder is reserved for bitmap/vector image assets (e.g. `logo.svg`, PNG exports).

Note: All UI icons used by the landing page are rendered as **inline SVG** inside
`script.js` (see the `Icons` object) to keep the page dependency-free and fast.
This folder exists to preserve a clean project structure and to hold any future
image assets you may want to reference from `config.json`.
